
 * 콜렉션 (표준화) //interface와 method가 overriding이 된다
 	- 기본적인 자료구조(개념)를 만들어줌
	- java.util 패키지

================
추가하는것들은 밑에걸로 다 되어있음
================
 1. List
 		- ArrayList : Object[] 동적크기
 		- 순서저장( index 보유 )
 
 2. Set
 		- 순서없음 ( index 없음 )
 		- 중복값 저장 불가
 		- 파이썬에선 많이 쓰임
 		
 3. Map
 		- 키(key)와 밸류(value) 저장 구조
 		- 키는 중복 안됨
